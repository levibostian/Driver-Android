abstract fun finishedFirstFetch(errorDuringFetch: Throwable?): Unit

Parameters

errorDuringFetch - Error that occurred during the fetch of getting first set of cache. It is up to you to capture this error and determine how to show it to the user. It's best practice that when there is an error here, you will dismiss a loading UI if you are showing one since finishedFirstFetch was called before.
